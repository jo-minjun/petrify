# Petrify

viwoods .note 파일을 Obsidian Excalidraw 포맷으로 변환하는 도구입니다.

## 패키지 구조

| 패키지 | 설명 |
|--------|------|
| `@petrify/core` | 핵심 모델, Excalidraw 변환, 포트 인터페이스 |
| `@petrify/parser-viwoods` | viwoods .note 파일 파서 |
| `obsidian-petrify` | Obsidian 플러그인 (예정) |

## 설치

```bash
pnpm install
```

## 빌드

```bash
pnpm build
```

## 테스트

```bash
pnpm test
```
